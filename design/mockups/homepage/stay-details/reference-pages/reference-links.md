## Reference 1: Plum Guide — "Parisian Gaze"
**URL:** https://www.plumguide.com/homes/1284787/parisian-gaze

### Key Design Patterns
- **Hero:** Full-width image carousel with thumbnail navigation below (14 photos)
- **Property header:** Serif headline (DomaineDisplayNarrow, 40-72px), guest/bedroom/bathroom bullets, breadcrumb navigation
- **Highlight cards:** 3 featured highlights with image overlays + descriptions (e.g., proximity, kitchen, bathroom)
- **Home Truths:** Bulleted icon+text list of key details (housekeeping, amenities, check-in)
- **Description:** Two-column layout — 75% editorial content / 25% sticky sidebar with contact/chat widget
- **Amenities:** Grid layout (54 amenities), "Show all" expandable CTA
- **Rooms:** Detailed bed type specs + bathroom features + floorplan visualization
- **Attractions:** Horizontal scrollable carousel — image + title + walking duration cards
- **Booking:** Sticky sidebar on desktop, fixed-bottom drawer on mobile — price, date/guest selectors, "Reserve" button
- **VIP Services:** Expandable list (concierge, chef, massage, etc.)
- **Host info:** Profile card with bio + award badge
- **Footer:** 5-6 column grid, dark background

### What to Adapt for Viatu
- The highlight cards approach (3 standout features with images) could work well for Viatu stays
- Attractions/local guide carousel is a great pattern for safari context (nearby parks, activities)
- Two-column description with sticky booking sidebar is solid UX
- "Home Truths" concept could become a "Good to Know" section for lodges

## Reference 2: Airbnb — Bergen Apartment
**URL:** https://www.airbnb.com/rooms/1420459357513907033

### Key Design Patterns
- **Gallery:** Responsive grid of 8 high-res images, adapts across breakpoints (744px, 950px, 1128px, 1440px)
- **Title area:** Property name + 4.93/5 stars (29 reviews) + location
- **Layout:** Two-column responsive grid — 58.33% content / 33.33% sticky booking sidebar
- **Typography:** Airbnb Cereal VF font, semibold titles (22-32px), medium body (14-16px), line-height 1.125-1.5rem
- **Colors:** Primary #222222, accent red #FF385C
- **Flow:** nav → imagery → title/rating → description → amenities → calendar → map → reviews → booking
- **Buttons:** 40px icon buttons with 50% border-radius, focus states, hover effects
- **Sticky sidebar:** `top: 80px` offset for booking widget visibility

### What to Adapt for Viatu
- The two-column content/booking layout is industry standard and works well
- Rating display with review count is a proven trust pattern
- Calendar/availability integration for booking flow
- Responsive image grid approach (but with Viatu's rounded corners aesthetic)

## Reference 3: Gmundner Lodge — Double Suite
**URL:** https://gmundner-lodge.com/gmundner-double-suite/

### Key Design Patterns
- **Hero:** Full-width landscape image with emotional overlay text ("When the world feels uncertain, choose a place that isn't")
- **Room header:** Large bold headline + structured spec grid (Guests: 2+1, Bedrooms: 1 Super King, Bathrooms: 1+outdoor, Size: 205 sqm)
- **Description:** ~150 word prose, serif body font (16-18px), left-aligned
- **Amenities ("Suite Features"):** Icon + text grid — monochromatic SVG icons (#424242, 40x40px) with labels (14px sans-serif)
- **Icons cover:** shower, bathtub, TV, yoga mats, safe, minibar, laundry, WiFi, AC, security
- **Footer:** Multi-column — brand logo + "Back to Nature" tagline, social links, nav columns, Relais & Châteaux partnership logo
- **Colors:** Warm neutrals (#424242 text), gold accents (#d3a170), light backgrounds (#f3f1ec)
- **Typography:** Bold sans-serif headlines, serif body text, uppercase labels

### What to Adapt for Viatu
- The emotional hero overlay text is very aligned with Viatu's inspirational tone
- Structured room specs grid (guests/beds/baths/size) is clean and scannable
- Monochromatic icon grid for amenities is elegant — could match Viatu's style with Phthalo green icons
- "Back to Nature" brand positioning parallels Viatu's sustainability focus
- Relais & Châteaux partnership badge concept could be used for Viatu's sustainability certifications
