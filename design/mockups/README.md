# Mockups

Place wireframes and design mockups here. Accepted formats:

- `.fig` — Figma files
- `.pdf` — exported design specs
- `.png` / `.jpg` — mockup screenshots per page or section

## Naming convention

```
home-hero.png
pricing-page.png
dashboard-overview.fig
onboarding-flow.pdf
```

Claude will read these before building the corresponding pages or sections.
