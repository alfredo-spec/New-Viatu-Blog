# Logos

Place all logo variants here. Recommended formats:

| File | Purpose |
|------|---------|
| `logo.svg` | Primary logo — full color |
| `logo-dark.svg` | Logo variant for dark backgrounds |
| `logo-light.svg` | Logo variant for light backgrounds |
| `logo-mark.svg` | Icon/mark only, no wordmark |
| `logo-wordmark.svg` | Wordmark only, no icon |
| `logo.png` | Raster fallback (2x resolution) |

Claude will use these files when building headers, footers, and brand moments.
Until real logos are provided, a clean SVG placeholder will be used automatically.
