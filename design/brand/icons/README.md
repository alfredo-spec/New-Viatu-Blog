# Icons

Place custom icon files here.

| File | Purpose |
|------|---------|
| `favicon.ico` | Browser tab icon |
| `favicon.svg` | SVG favicon (modern browsers) |
| `apple-touch-icon.png` | iOS home screen icon (180×180px) |
| `og-image.png` | Default Open Graph image (1200×630px) |

Custom UI icons can also be stored here as SVGs and imported as Vue components.
