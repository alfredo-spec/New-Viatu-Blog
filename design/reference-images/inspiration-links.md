# Inspiration Links — Viatu Blog Design References

> **Usage note for Claude:** Before building or modifying any UI component, page, or layout,
> scan this document for URLs relevant to the current task. Use **WebFetch** to study
> 2–3 of the most relevant sites, noting specific patterns to **adapt** (never copy).
> Focus on layout structure, typography hierarchy, image-text rhythm, whitespace usage,
> and navigation patterns. Always filter observations through the Viatu design system
> defined in `/design/style-guides/viatu-design-system.md`.
