# Style Guides

Place CSS style guide files here. These define the rules Claude must follow when writing any UI code for this project.

## Expected files (add as the project evolves)

- `typography.md` or `typography.css` — font families, sizes, weights, line heights, tracking
- `spacing.md` or `spacing.css` — spacing scale, grid, margins, padding rules
- `colors.md` or `colors.css` — palette, semantic color tokens, dark/light mode
- `components.md` — button styles, card styles, form element rules
- `motion.md` — transition durations, easing curves, animation principles
